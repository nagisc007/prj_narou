**「異世界に行けば何とかなる、そう思っていた頃がありました」**

異世界に転生した主人公は、いきなり死んでしまう。次に目を開けるとそこは更に別の異世界だった。

## Description

異世界に転生した主人公は、突然目の前が暗くなり、死んでしまう。  
しかし再び目覚める主人公。だがそこはまた別の異世界だった。  
次々と遅いくる「本物の異世界の残酷」  
主人公は記憶と容姿だけを引き継いで、次々と新しい異世界に転生する謎の旅を繰り返す。  
何度も死にながら、自分は何もしてこなかった、ただ望んでいただけだった現実に目を向けて、  
少しずつ生き延びる時間を増やしていく。  
そしてある異世界に辿り着いた時、そこには自分のことを知る女性が存在した。

## Characters

- 悪鞍汚悪……何をしてもついていない、文句ばかりの元無職
- 水無本美綯……主人公が憧れていた同級生
- ゼル……次元の妖精と自称する謎の少女

## Features

※徐々に文字数が増える（＝その世界で長く生きることができた証拠）仕様。

### World.0 「始まりと終わりの０の執行人」

（予定0.1K）

あ。  
だけの言葉を残して、主人公は死んでしまう。


### World.1 「１からやり直す異世界生活」

（予定1K）

目覚めると知らない町が広がっていた。  
そこは獣人やエルフも一緒に暮らす異世界、のようにも見えた。  
異世界転生したのだと理解したが、突然路地で襲われ成すすべもなく殺される主人公だった。

### World.2 「この素晴らしい２番煎じの異世界に祝福を」

（予定10K）

目覚めるとまた違う世界。  
ギルドと呼ばれるものがあり、冒険者となったり、まるでＲＰＧの世界に似ている。  
だが何をするにも金が必要だった。  
しかし主人公は無一文で、何もできない。  
そんな主人公だけが見える存在がいた。  
彼女は自分を次元の妖精と名乗るが、その妖精から転生得意体という特別な存在となったことを明かされる。  
主人公は何とか以前の記憶を取り戻して、この世界で生き抜こうとするのだが、金を得ることができず、  
日に日に痩せ衰えて、最後には強盗に殺された。

### World.3 「サード・ロボット・オンライン」

（予定20K）

目覚めるとゼルを確認し、この世界についての知識を得る。  
ここはVRMMO的な世界観らしい。  
今までのように自分を殺しに来ないことに安堵するが、  
ある日、この世界のマスターとやらが出現し、デスゲーム宣言がなされる。  
それを境に人々は殺し合いを始めてしまう。  
そんな中で出会った不思議な力を持つ少女は、何と彼の初恋の同級生だった。  
彼女もまた転生得意体となったようだったが、その彼女が目の前で殺される姿を目撃する。  
怒りに任せて行動した結果、主人公もまた爆炎に呑み込まれて死んだ。

### World.4 「フォーエバー・ロード」

（予定40K）

目覚めるとゼルがいない。  
そして今までとは違い、自分を慕う多くの部下がいることに気づく。  
そこはどうやらVRMMORPGによく似た異世界のようで、  
更に自身は魔王に相当するチート能力者だと判明する。  
そして自分を慕う部下の中に、彼女がいた。  
ゼルの助けがない中、部下たちと共にこの世界で何が起こっているのかを調査する。  
だが、部下の一人に裏切られ、窮地に陥る。  
その時、いつも最悪を引き寄せると言われたことで、記憶の一部が蘇る。  
何故自分は転生しているのか。  
何故彼女は転生しているのか。  
その謎を思い出す前に、ドラゴンの炎により焼かれてしまった。

### World.5 「ゴーアヘッド・ゴアライフ」

（予定50K）※ここを書けば規定量クリア

目覚めると消えていたはずのゼルがいる。  
そこは全てがゲームで決まる世界。  
ゲーム強者だけが生き残ることができる。  
その世界で賞金首となった主人公は、突如現れた彼女に助けられる。  
何故か彼女と共に大会に参加することになる。  
大会ではその時々で出されるゲームが異なり、知力だけでなく、技術や知識も問われる。  
その大会で勝ち進む毎に、記憶の一部が蘇っていく。  
そのうちに、学生時代の記憶がおぼろげに蘇る。  
彼女は約束してくれたことを思い出す。  
「あなたがどんな世界にいっても、守ってあげる」  
そのことを彼女は無意識で守っていたのだ。  
今の世界が大嫌いだった主人公。  
違う世界ならやり直せると言った彼に、そんなことないと諭してくれた彼女。  
これが夢なら醒めてくれと、主人公はゲームエンドに飛び込んだ。

### World.6 「ロク・サンライズ」

（予定60K）

目覚めると、そこは普通の住宅だった。  
何故か小さい頃に亡くなったはずの両親もいる世界。  
しかし街に出ると、そこにはVRMMORPGに出るような冒険者や怪物が闊歩していた。  
異世界が現実世界に侵食したのだと言われ、どうすれば元の世界に戻せるのか考える。  
やがて彼女と協力しながら自分たちの拠点を守り、  
そこで暮らしていくという安住を求めるようになる。  
だがある時、別の異世界が侵略してきた。  
それは巨大ロボットが登場するSFの世界観。  
やがてその戦争に巻き込まれ、みんな死んでしまう。

### World.7 「七宮ナツヒの消失」

（予定70K）

目覚めると、そこは知らない会議室だった。  
そして学校で、学生時代に戻っていた。  
主人公はこれまでずっと一緒に転生してきたはずのナツヒを探す。  
しかし誰も彼女を知らない。  
一体どうなっているのか。  
この世界、そして転生世界の謎を解く為に、ゼルを知るという少女「永田」と共に行動する。

### World.8 「魔法少女本気花エイト」

（予定80K）

目覚めると知らない学校だった。  
と、突然悲鳴が上がる。  
少女が何か知らないものに襲われていた。  
それを見たゼルは「魔女」だという。  
そして魔法少女となったナツヒが魔女を倒す。  
しかし魔力を使うと消耗し、やがて魔女になるという。  
その消耗を癒してあげられるのが主人公だけと知り、  
彼は彼女の為に生命力を捧げる。  
やがて彼の生命力が果て、彼が魔王となる。  
その中で、彼は全てを思い出す。  
自分が世界に、他人に、大量の憎悪を持っていたことを。  

### World.9 「九界戦線」

（予定90K）

目覚めるとそこは、彼がかつてノートに書いていた有象無象の異世界だった。  
自分が世界の壁を破壊してしまったことを知り、何とか元に戻そうと奔走する。  


### World.10 「テンペスト　彼の地に出会いを求めるのは間違っていた」

（予定100K）

目覚めると、そこは暗黒の世界。  
ゼルは全ての終わりにして始まりの世界だという。
